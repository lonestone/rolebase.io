## Vue d'ensemble

Le fil d'actualité est votre **tableau de bord central** pour suivre ce qui se passe dans l'organisation. Il agrège l'activité récente afin que vous n'ayez pas à vérifier chaque rôle individuellement.

## Ce qui apparaît dans le fil d'actualité

Le fil d'actualité affiche l'activité récente, notamment :

- **Discussions nouvelles et mises à jour** : discussions qui ont été créées ou qui ont reçu de nouveaux messages.
- **Décisions** : décisions formelles enregistrées dans tout rôle auquel vous avez accès.
- **Réunions terminées** : réunions qui se sont terminées, avec un lien vers leur résumé.

Les éléments sont affichés dans l'ordre chronologique inverse, avec l'activité la plus récente en haut.

## Vue d'ensemble du tableau de bord

Le tableau de bord est le premier écran que vous voyez lorsque vous ouvrez Rolebase. En plus du fil d'actualité, il fournit des panneaux d'accès rapide pour :

### Mes tâches

Une vue consolidée de toutes les tâches qui vous sont assignées dans tous les rôles. Voyez rapidement ce qui nécessite votre attention sans naviguer vers les rôles individuels.

### Réunions à venir

Une liste de vos prochaines réunions planifiées avec la date, l'heure et le rôle. Cliquez sur n'importe quelle réunion pour voir ses détails ou rejoindre la session.

### Discussions actives

Les discussions auxquelles vous participez et qui ont une activité récente. Cela vous aide à suivre les échanges en cours.

<Callout type="tip" title="Navigation rapide">
  Chaque élément du tableau de bord renvoie directement vers sa vue détaillée. Cliquez sur une tâche, une réunion ou une discussion pour y accéder directement.
</Callout>

## Paramètres de notification

Rolebase peut envoyer des **e-mails récapitulatifs** pour vous tenir informé lorsque vous n'êtes pas sur l'application. Configurez vos préférences de notification pour contrôler :

- La **fréquence** détermine à quel rythme vous recevez les e-mails récapitulatifs (quotidien, hebdomadaire ou désactivé).
- Le **contenu** détermine quels types d'activité sont inclus dans le récapitulatif.

Les e-mails récapitulatifs résument les discussions, décisions et réunions récentes afin que vous puissiez rester informé sans être submergé.

<Callout type="info" title="Notifications par e-mail">
  Les préférences de notification sont par utilisateur et peuvent être modifiées depuis les paramètres de votre compte. Elles s'appliquent à toutes les organisations auxquelles vous appartenez.
</Callout>

## Visibilité et confidentialité

Le fil d'actualité respecte les paramètres de confidentialité :

- **Rôles privés** : l'activité des rôles privés n'est affichée qu'aux membres du rôle.
- **Discussions et décisions privées** : les éléments privés n'apparaissent que pour les participants autorisés.
- **Réunions privées** : visibles uniquement par les participants à la réunion.

Vous ne verrez que l'activité des rôles et éléments auxquels vous avez accès.
