# Rolebase Documentation

**[Rolebase](https://rolebase.io)** is the online headquarter for [liberated companies](https://en.wikipedia.org/wiki/Liberated_company). It's an open-source SaaS that helps organizations implement Holacracy, Sociocracy, and other forms of self-governance.

<Callout type="tip" title="New to Rolebase?">
  Start with the [Introduction](/en/docs) to understand the core concepts, then follow the [Getting Started](/en/docs/getting-started) guide.
</Callout>
