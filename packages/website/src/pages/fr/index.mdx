# Documentation Rolebase

**[Rolebase](https://rolebase.io)** est le quartier général en ligne des [entreprises libérées](https://fr.wikipedia.org/wiki/Entreprise_lib%C3%A9r%C3%A9e). C'est un SaaS open-source qui aide les organisations à mettre en place l'Holacratie, la Sociocratie et d'autres formes d'auto-gouvernance.

<Callout type="tip" title="Nouveau sur Rolebase ?">
  Commencez par l'[Introduction](/fr/docs) pour comprendre les concepts clés, puis suivez le guide [Premiers pas](/fr/docs/getting-started).
</Callout>
