## Overview

The interactive org chart is the primary way to visualize your organizational structure. It displays roles as nested containers, where a role that contains sub-roles represents a team or department. You can:

- **Click** a role to zoom in and see its sub-roles.
- **Drag** to pan around the org chart.
- **Scroll** to zoom in and out.
- Switch between views using the view selector.

## Views

The org chart offers four different views. You can set the default view in the [organization settings](/en/docs/organizations), and switch between views at any time using the view selector.

| View | Description |
|---|---|
| **All Roles** | Shows the full hierarchy of all roles and sub-roles. The size of each role depends on the number of sub-roles and members it contains. |
| **Holarchy** | Displays the sub-roles of the selected role at the same size, with the members who represent them. |
| **Operational** | Displays all roles at the same level, mixed, with the members who represent them. |
| **Members only** | Displays the organization from a member-centric perspective. |

<Callout type="tip" title="Navigation">
  Regardless of the default view, users can always switch between views and click into any role to explore its contents.
</Callout>
