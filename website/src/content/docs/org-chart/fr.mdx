## Présentation

L'organigramme interactif est le moyen principal de visualiser votre structure organisationnelle. Il affiche les rôles sous forme de conteneurs imbriqués, où un rôle contenant des sous-rôles représente une équipe ou un département. Vous pouvez :

- **Cliquer** sur un rôle pour zoomer et voir ses sous-rôles.
- **Glisser** pour naviguer dans l'organigramme.
- **Défiler** pour zoomer et dézoomer.
- Basculer entre les vues à l'aide du sélecteur de vue.

## Vues

L'organigramme propose quatre vues différentes. Vous pouvez définir la vue par défaut dans les [paramètres de l'organisation](/fr/docs/organizations), et basculer entre les vues à tout moment à l'aide du sélecteur.

| Vue                    | Description                                                                                                                                           |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Tous les Rôles**     | Affiche la hiérarchie complète de tous les rôles et sous-rôles. La taille de chaque rôle dépend du nombre de sous-rôles et de membres qu'il contient. |
| **Holarchie**          | Affiche à la même taille les sous-rôles du rôle sélectionné, avec les membres qui les représentent.                                                   |
| **Opérationnel**       | Affiche tous les rôles à un même niveau, mélangés, avec les membres qui les représentent.                                                             |
| **Membres uniquement** | Affiche l'organisation du point de vue des membres.                                                                                                   |

<Callout type="tip" title="Navigation">
  Quelle que soit la vue par défaut, les utilisateurs peuvent toujours basculer
  entre les vues et cliquer sur n'importe quel rôle pour explorer son contenu.
</Callout>
